# AGENTS.md

## 프로젝트 목표
한국어 사용자를 위한 MBTI 스타일 성향 테스트 웹사이트를 만들고, 모바일 우선 UX와 Google AdSense 수익화를 함께 고려한다.

## 제품 제약
- 사용자는 24개 문항에 응답해야 한다.
- 결과는 16개 성격 유형 중 하나로 귀결되어야 한다.
- 광고는 결과 페이지 또는 콘텐츠 페이지에만 노출할 수 있다.
- 질문 진행 중인 플로우에는 광고를 배치하지 않는다.
- 앱은 모바일에서 가볍고 빠르게 느껴져야 한다.

## 엔지니어링 규칙
- TypeScript를 엄격하게 사용한다.
- Next.js를 사용한다.
- 스코어링 로직은 UI 컴포넌트 밖에 둔다.
- 점수 계산과 결과 판정은 가능한 순수 함수로 작성한다.
- 질문 데이터는 JSON 또는 명시적 타입 구조로 관리한다.
- UI는 재사용 가능한 컴포넌트 중심으로 만들고 중복 로직을 피한다.
- 클라이언트 상태가 꼭 필요한 경우를 제외하고 서버 컴포넌트를 우선한다.

## 품질 규칙
작업 완료 전 반드시 아래 항목을 모두 통과해야 한다.
- lint
- typecheck
- tests
- build

## SEO 및 UX
- 결과 페이지는 의미 있는 메타데이터를 가져야 한다.
- 모바일 레이아웃은 과도한 광고 없이 읽기 쉬워야 한다.
- 주요 CTA와 결과 요약은 쉽게 찾을 수 있어야 한다.
